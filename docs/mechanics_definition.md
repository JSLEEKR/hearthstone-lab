# 하스스톤 메커니즘 정의서

> 전체 67개 메커니즘 + 13개 종족 + 7개 주문 학파 정의

---

## 1. 전투 키워드 (Combat Keywords)

게임 엔진에서 직접 처리해야 하는 전투 관련 키워드.

### TAUNT (도발) - 477장
- **효과**: 이 하수인이 있으면 상대는 이 하수인만 공격 가능
- **규칙**: 도발 하수인이 여러 개면 그 중 하나 선택 가능. 은신+도발이면 은신이 우선 (공격 대상 지정 불가)
- **해제**: 침묵(SILENCE)으로 제거 가능
- **구현 상태**: ✅ 완료 (get_legal_actions에서 처리)

### DIVINE_SHIELD (천상의 보호막) - 93장
- **효과**: 처음 받는 피해를 무효화
- **규칙**: 0 피해는 보호막을 소모하지 않음. 독성 피해도 보호막이 흡수 (독성 무효)
- **해제**: 피해 1회 흡수 후 제거, 침묵으로 제거
- **구현 상태**: ✅ 완료 (take_damage에서 처리)

### CHARGE (돌격) - 46장
- **효과**: 소환된 턴에 바로 공격 가능 (영웅/하수인 모두)
- **규칙**: summoned_this_turn이어도 공격 가능
- **구현 상태**: ✅ 완료 (can_attack에서 처리)

### RUSH (돌진) - 238장
- **효과**: 소환된 턴에 하수인만 공격 가능 (영웅 공격 불가)
- **규칙**: 다음 턴부터는 영웅도 공격 가능
- **구현 상태**: ✅ 완료 (can_attack_hero에서 처리)

### WINDFURY (바람의 분노) - 35장
- **효과**: 한 턴에 2번 공격 가능
- **규칙**: attacks_this_turn < 2일 때 공격 가능
- **해제**: 침묵으로 제거 (남은 공격 횟수는 유지)
- **구현 상태**: ✅ 완료 (can_attack에서 처리)

### STEALTH (은신) - 83장
- **효과**: 상대가 공격 대상으로 지정 불가
- **규칙**: 공격하면 은신 해제. 도발+은신이면 도발 무시됨 (공격 불가하므로). 전체 피해(AOE)는 은신 무시하고 적용
- **해제**: 공격 시, 침묵 시
- **구현 상태**: ⚠️ 부분 완료 (영웅 공격 시만 해제, 하수인 공격 시 해제 누락)

### FREEZE (빙결) - 30장
- **효과**: 빙결된 캐릭터는 다음 턴에 공격 불가
- **규칙**: 빙결 상태의 하수인은 자신의 턴 종료 시 빙결 해제. FREEZE 메커니즘이 있는 하수인/주문이 피해를 입히면 대상 빙결
- **적용 시점**: 피해를 입힐 때 (전투, 주문 등)
- **구현 상태**: ⚠️ 부분 완료 (해동만 구현, 빙결 적용 미구현)

### POISONOUS (독성) - 29장
- **효과**: 피해를 입힌 하수인을 즉사시킴
- **규칙**: 0보다 큰 피해를 입혀야 함. 천상의 보호막이 피해를 흡수하면 독성 무효. 독성 vs 독성이면 둘 다 즉사
- **구현 상태**: ❌ 미구현

### LIFESTEAL (생명력 흡수) - 133장
- **효과**: 피해를 입힌 만큼 아군 영웅 회복
- **규칙**: 영웅 최대 체력(30) 초과 회복 불가. 전체 피해 시 모든 피해 합산하여 회복
- **구현 상태**: ❌ 미구현

### REBORN (환생) - 31장
- **효과**: 처음 사망 시 체력 1로 부활
- **규칙**: 환생으로 부활한 하수인은 환생 키워드 없음 (1회만). 보드 꽉 차면 부활 불가. 사망한 위치에 부활
- **구현 상태**: ❌ 미구현

### CANT_ATTACK (공격 불가) - 15장
- **효과**: 이 하수인은 공격할 수 없음
- **규칙**: 구제 능력이 있어도 공격 불가
- **구현 상태**: ❌ 미구현 (can_attack에서 체크 필요)

### ELUSIVE (회피) - 38장
- **효과**: 주문이나 영웅 능력의 대상이 될 수 없음
- **규칙**: 전체 대상 효과(AOE)는 적용됨. 아군 대상 주문도 불가
- **구현 상태**: ❌ 미구현 (주문 대상 선택 시 필터 필요)

### ENRAGED (격노) - 18장
- **효과**: 피해를 입은 상태(체력 < 최대체력)일 때 추가 효과
- **규칙**: 회복되면 효과 해제. 보통 공격력 증가
- **구현 상태**: ❌ 미구현

## 2. 카드 사용 트리거 (Play Triggers)

카드를 사용할 때 발동하는 효과.

### BATTLECRY (전투의 함성) - 2,236장
- **효과**: 손에서 낼 때 발동 (소환/복사는 미발동)
- **규칙**: 대상 지정이 필요한 경우 유효 대상이 없으면 그냥 소환만. 변형(Choose One 등)과 결합 가능
- **발동 시점**: 하수인이 보드에 놓인 직후
- **구현 상태**: ❌ 미구현 (소환만 됨, 효과 없음)

### COMBO (연계) - 83장
- **효과**: 이번 턴에 다른 카드를 먼저 사용했으면 추가 효과
- **규칙**: 턴 시작 후 1장이라도 사용했으면 발동. 동전(COIN)도 카드로 취급
- **구현 상태**: ❌ 미구현

### OUTCAST (추방자) - 33장
- **효과**: 손의 맨 왼쪽 또는 맨 오른쪽에서 사용 시 추가 효과
- **규칙**: 악마사냥꾼 전용
- **구현 상태**: ❌ 미구현

### CHOOSE_ONE (선택) - 101장
- **효과**: 2~3개 옵션 중 하나 선택
- **규칙**: 드루이드 전용 (일부 중립). "두 가지 모두" 효과(판다르 등)와 결합
- **구현 상태**: ❌ 미구현

### OVERLOAD (과부하) - 81장
- **효과**: 이번 턴에 추가 마나를 쓰고, 다음 턴에 과부하된 마나만큼 사용 불가
- **규칙**: 과부하량은 카드별 고정. 중첩 가능. "과부하: (2)"면 다음 턴 마나 -2
- **데이터**: card_data의 `overload` 또는 text에서 파싱 ("과부하: (N)")
- **구현 상태**: ⚠️ 마나 차감은 구현, 과부하 적용 미구현

## 3. 죽음/파괴 트리거 (Death Triggers)

### DEATHRATTLE (죽음의 메아리) - 670장
- **효과**: 하수인 사망 시 발동
- **규칙**: 선소환 순서대로 발동. 침묵 시 제거. 변형으로 사망해도 발동. 무기의 죽음의 메아리는 무기 파괴 시 발동
- **발동 시점**: 모든 동시 사망 처리 후
- **구현 상태**: ❌ 미구현

### HONORABLE_KILL (명예로운 처치) - 20장
- **효과**: 정확히 치명타로 처치 시 (체력을 정확히 0으로 만들 때) 추가 효과
- **규칙**: 초과 피해 시 미발동 (체력이 음수가 되면 안 됨... 실제로는 0 이하면 발동)
- **참고**: 실제 게임에서는 "남은 체력 이하의 피해로 처치"가 아니라 "이 카드의 피해로 처치"면 발동
- **구현 상태**: ❌ 미구현

## 4. 지속 효과 (Persistent Effects)

### AURA (오라/지속 효과) - 171장
- **효과**: 보드에 있는 동안 지속적으로 효과 적용
- **예시**: "다른 아군 하수인에게 +1 공격력", "아군 주문 비용 (1) 감소"
- **규칙**: 하수인이 사라지면 효과도 사라짐. 매 상태 체크 시 재계산
- **구현 상태**: ❌ 미구현

### TRIGGER_VISUAL (트리거 시각 효과) - 936장
- **효과**: 특정 조건 충족 시 발동하는 효과 (시각적 표시용 태그)
- **예시**: "아군 하수인이 사망할 때마다 +1/+1", "카드를 뽑을 때마다 피해 1"
- **규칙**: 보드에 있는 동안만 발동. 실제 트리거 로직은 text에서 파싱 필요
- **구현 상태**: ❌ 미구현

### SPELLPOWER (주문 공격력) - 57장
- **효과**: 아군 피해 주문에 추가 피해 부여
- **규칙**: 보드에 있는 동안 적용. 중첩 가능. AFFECTED_BY_SPELL_POWER 태그가 있는 주문만 영향
- **구현 상태**: ❌ 미구현

### SECRET (비밀) - 117장
- **효과**: 상대 턴에 특정 조건 충족 시 자동 발동
- **규칙**: 한 번에 같은 비밀 중복 불가. 발동 순서는 사용 순서. 비밀 개수 제한 (5개)
- **직업별**: 마법사(3마나), 사냥꾼(2마나), 성기사(1마나), 도적(2마나)
- **구현 상태**: ❌ 미구현

## 5. 카드 생성/변형 키워드 (Generation Keywords)

### DISCOVER (발견) - 367장
- **효과**: 3장 중 1장 선택
- **규칙**:
  - 기본 풀: 자신의 직업 + 중립 카드 (표준/야생은 현재 게임 포맷 따름)
  - 특정 필터가 있으면 해당 조건 적용 ("용 발견", "주문 발견" 등)
  - 3장은 서로 중복 없음
  - 선택한 카드가 손에 추가됨
  - **보너스**: 자신의 직업 카드가 4배 확률로 등장
- **풀 예시**:
  - "주문 발견" → 자신의 직업 주문 + 중립 주문
  - "용 발견" → 종족이 DRAGON인 하수인
  - "죽음의 기사 카드 발견" → DEATHKNIGHT 직업 카드
- **구현 상태**: ❌ 미구현

### ECHO (메아리) - 14장
- **효과**: 사용 후 임시 복사본이 손에 추가됨 (같은 턴에만)
- **규칙**: 턴 종료 시 복사본 제거. 마나가 있는 한 계속 사용 가능
- **구현 상태**: ❌ 미구현

### TWINSPELL (쌍둥이 주문) - 11장
- **효과**: 처음 사용 시 쌍둥이 주문 없는 복사본이 손에 추가
- **규칙**: 복사본은 TWINSPELL 키워드 없음 (1회만)
- **구현 상태**: ❌ 미구현

### CORRUPT (타락) - 31장
- **효과**: 손에 있을 때 더 비싼 카드를 사용하면 타락(강화)됨
- **규칙**: 타락 후 텍스트가 변경됨. 비용이 "더 비싼" 카드만 (같거나 싼 카드는 안 됨)
- **구현 상태**: ❌ 미구현

### INFUSE (주입) - 52장
- **효과**: 손에 있을 때 아군 하수인 N체 사망 시 강화
- **규칙**: 사망 수 카운트가 조건 충족하면 카드 변환
- **구현 상태**: ❌ 미구현

### TRADEABLE (교환 가능) - 49장
- **효과**: 1마나를 내고 이 카드를 덱에 섞고 새 카드 드로우
- **규칙**: 교환은 "카드 사용"으로 취급 안 됨 (연계 미발동)
- **구현 상태**: ❌ 미구현

### FORGE (벼림) - 20장
- **효과**: 손에서 2마나를 내고 강화된 버전으로 변환
- **규칙**: 강화는 영구적. 턴에 여러 번 가능
- **구현 상태**: ❌ 미구현

### MINIATURIZE (소형화) - 19장
- **효과**: 소환 시 1마나 1/1 복사본도 함께 소환
- **규칙**: 보드에 자리가 없으면 소형 복사본 미소환
- **구현 상태**: ❌ 미구현

### MAGNETIC (자석) - 26장
- **효과**: 아군 기계 하수인 왼쪽에 놓으면 합체 (스탯과 효과 부여)
- **규칙**: 합체 시 대상 하수인에 공격력/체력/키워드 추가. 왼쪽에 놓지 않으면 일반 소환
- **구현 상태**: ❌ 미구현

## 6. 특수 확장팩 키워드 (Expansion-Specific)

### QUEST (퀘스트) - 30장
- **효과**: 게임 시작 시 자동 멀리건 유지, 조건 충족 시 보상
- **규칙**: 덱에 1장만 가능. 사이드퀘스트(SIDE_QUEST)는 일반 카드처럼 사용
- **관련**: SIDE_QUEST (9장)
- **구현 상태**: ❌ 미구현

### DREDGE (준설) - 21장
- **효과**: 덱 아래 3장 중 1장을 덱 위로 올림
- **규칙**: 덱이 3장 미만이면 있는 만큼만
- **구현 상태**: ❌ 미구현

### COLOSSAL (거대) - 24장
- **효과**: 소환 시 추가 부위(appendage) 하수인도 함께 소환
- **규칙**: "거대 +N"으로 추가 소환 수 결정. 보드 자리 부족 시 가능한 만큼만
- **구현 상태**: ❌ 미구현

### TITAN (타이탄) - 12장
- **효과**: 소환 시 3가지 능력 중 매 턴 1개씩 선택 사용 (총 3턴)
- **규칙**: 3개 능력 모두 사용 후 일반 하수인처럼 공격 가능. 사용 순서 자유
- **구현 상태**: ❌ 미구현

### EXCAVATE (발굴) - 17장
- **효과**: 발굴 진행도 증가 + 보물 발견
- **규칙**: 1→2→3→4단계로 진행, 단계별 보물 품질 상승. 직업별 고유 보물
- **구현 상태**: ❌ 미구현

### INSPIRE (감화) - 27장
- **효과**: 영웅 능력 사용 시 발동
- **규칙**: 보드에 있는 동안만 발동
- **구현 상태**: ❌ 미구현

### SPELLBURST (주문폭발) - 43장
- **효과**: 주문 사용 시 1회 발동 후 소멸
- **규칙**: 아군 주문만. 1회 발동 후 키워드 제거
- **구현 상태**: ❌ 미구현

### FRENZY (광란) - 18장
- **효과**: 처음 피해를 입을 때 1회 발동
- **규칙**: 1회만 발동. 피해가 0이면 미발동
- **구현 상태**: ❌ 미구현

### MANATHIRST (마나 갈증) - 18장
- **효과**: 마나 결정이 N개 이상이면 추가 효과
- **규칙**: max_mana 기준 (현재 마나가 아닌 최대 마나)
- **구현 상태**: ❌ 미구현

### QUICKDRAW (속사) - 16장
- **효과**: 뽑은 턴에 사용하면 추가 효과
- **규칙**: 드로우한 턴에만 발동. 멀리건에서 받은 카드도 해당
- **구현 상태**: ❌ 미구현

### STARSHIP_PIECE (우주선 부품) - 16장
- **효과**: 죽음의 메아리로 우주선에 부품 장착
- **규칙**: 거대한 어둠 너머 확장팩 전용. 부품 3개 장착 시 우주선 발사
- **구현 상태**: ❌ 미구현

### OVERKILL (과잉살상) - 15장
- **효과**: 대상을 초과 피해로 처치 시 추가 효과
- **규칙**: 자신의 턴에만 발동
- **구현 상태**: ❌ 미구현

### START_OF_GAME_KEYWORD (게임 시작 효과) - 11장
- **효과**: 게임 시작 시 덱 조건 충족하면 효과 적용
- **예시**: "덱에 짝수 비용 카드만 있으면 영웅 능력 비용 1" (바쿠), "홀수 비용만 있으면 영웅 능력 업그레이드" (겐)
- **구현 상태**: ❌ 미구현

### OVERHEAL (과잉 치유) - 10장
- **효과**: 최대 체력 이상으로 치유 시 추가 효과
- **규칙**: 초과 치유량 기반이 아님, 최대 체력인 상태에서 치유하면 발동
- **구현 상태**: ❌ 미구현

### JADE_GOLEM / JADE_LOTUS (비취 골렘) - 4+6장
- **효과**: 비취 골렘 소환 (1/1부터 시작, 소환할 때마다 +1/+1 성장)
- **규칙**: 최대 30/30. 게임 내 카운터 공유
- **구현 상태**: ❌ 미구현

## 11. 랜덤 생성 풀 규칙

카드 효과에서 "랜덤" 카드를 생성할 때의 풀 정의:

### 기본 규칙
- **포맷 제한**: 스탠다드 게임이면 스탠다드 카드만, 와일드면 전체
- **수집 가능만**: collectible=True인 카드만
- **직업 제한**: 자신의 직업 + 중립 (명시되지 않는 한)
- **토큰 제외**: collectible=False인 토큰은 생성 풀에 포함 안 됨

### 발견(Discover) 풀 규칙
- 3장 제시, 중복 없음
- 자신의 직업 카드 4배 가중치
- 텍스트의 조건에 따라 필터 추가

### 랜덤 소환 풀 규칙
- "무작위 N코스트 하수인" → mana_cost=N, card_type=MINION, collectible=True
- "무작위 야수" → card_type=MINION, race=BEAST
- "무작위 전설 하수인" → rarity=LEGENDARY, card_type=MINION
- "무작위 비밀" → mechanics에 SECRET 포함, hero_class=현재 직업

### 주요 생성 패턴 정리

| 패턴 | 필터 조건 |
|------|----------|
| 무작위 하수인 | type=MINION, collectible |
| 무작위 주문 | type=SPELL, collectible |
| 무작위 야수 | type=MINION, race=BEAST |
| 무작위 용 | type=MINION, race=DRAGON |
| 무작위 악마 | type=MINION, race=DEMON |
| 무작위 기계 | type=MINION, race=MECHANICAL |
| 무작위 멀록 | type=MINION, race=MURLOC |
| 무작위 정령 | type=MINION, race=ELEMENTAL |
| 무작위 해적 | type=MINION, race=PIRATE |
| 무작위 언데드 | type=MINION, race=UNDEAD |
| 무작위 나가 | type=MINION, race=NAGA |
| 무작위 토템 | type=MINION, race=TOTEM |
| 무작위 N코스트 하수인 | type=MINION, mana_cost=N |
| 무작위 N코스트 이하 하수인 | type=MINION, mana_cost<=N |
| 홀수 비용 카드 | mana_cost % 2 == 1 |
| 짝수 비용 카드 | mana_cost % 2 == 0 |
| 무작위 전설 하수인 | type=MINION, rarity=LEGENDARY |
| 직업 주문 발견 | type=SPELL, hero_class=CLASS (4x) + NEUTRAL |
| 직업 카드 발견 | hero_class=CLASS (4x) + NEUTRAL |
| 죽음의 메아리 하수인 | type=MINION, mechanics에 DEATHRATTLE |
| 전투의 함성 하수인 | type=MINION, mechanics에 BATTLECRY |
| 도발 하수인 | type=MINION, mechanics에 TAUNT |
| 돌진 하수인 | type=MINION, mechanics에 RUSH |
